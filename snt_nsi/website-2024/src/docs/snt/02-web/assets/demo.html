Page de démonstration

Bienvenue sur la page de démonstration du langage HTML.
Cette page est volontairement très simple afin que vous puissiez facilement observer son code source (c'est-à-dire le code informatique qui a permis de construire cette page).

Vous pouvez visualiser le code source d'une page web en utilisant l'une des techniques suivantes :

  • Utiliser le raccourci clavier CTRL + U (vue simple du code source)
  • Faire un clic droit sur la page puis « inspecter » (outil d'inspection)
  • Utiliser le raccourci clavier CTRL + MAJ + I (outil d'inspection)
